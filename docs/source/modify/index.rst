微调与修订
-----------

本章将介绍特定场景下，对运行图进行一定调整的相关功能。原则上，这些功能基本上都可以通过使用 :doc:`/data/index` 中介绍的基本编辑方法实现，但有的调整可能涉及到大量重复的工作。对于一些编写程序中已经考虑到的、比较常用的调整场景，本软件已经设计好了一些快速调整的功能，以减少重复的工作量。本章将说明这些功能。

显然，调整数据的场景千千万，软件不可能提前考虑好所有的情形。如果用户使用过程中发现一些没有实现的功能并认为有一定的价值，欢迎提出反馈。

本章的内容组织参照原pyETRC文档中相应章节，按照有关功能调整的主要数据对象分类。如果没有特殊说明，这里的操作都是支持撤销的。
