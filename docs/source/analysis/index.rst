运行图分析
----------

qETRC在提供基本的运行图铺画、数据管理功能之外，还提供了一系列运行图分析功能，用以对已有的运行图做统计，或提取特定的数据等。除了pyETRC时代就存在的标尺对照、区间对数表等功能外，qETRC还提供了大量新的分析功能，例如时刻诊断、间隔分析，也基于新的数据结构，提高了部分分析的严谨性和可靠性（如车次事件表）。

当然，运行图蕴含的数据是丰富的，从中可以提取的数据、可以做的分析也是非常庞杂的。由于实现难度、作者的时间精力、背景知识和开发能力等的限制，qETRC的分析功能自然还是挂一漏万。当然，欢迎提供建设性的建议。

.. important::
    qETRC中分析统计数据时，对列车间隔等概念做了定义或规定。这些定义或规定仅仅是qETRC软件的规定，在软件内部尽量保持自洽性。但这些定义不代表特定领域内学界或业界的定义，使用本软件分析时，请注意数据的解释。
